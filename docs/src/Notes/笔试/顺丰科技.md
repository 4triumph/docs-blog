## 笔试
```javascript
const file = 'https://baidu.com/word.png';
const ans = [];
let n = file.length;
for(let i = n - 1; i >= 0; i -- ){
  if(file[i] == '/'){
    break;
  }
  ans.push(file[i]);

}
ans.reverse();
console.log(ans.join(''))  //可恶的join
```
